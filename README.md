# Market Manager

## Overview

Development of a market management tool focused on simplifying sales tracking, inventory management, and financial record keeping for small-scale sellers.

The project aims to provide a practical solution for home-based or micro-business ventures that currently rely on manual methods such as notebooks or spreadsheets to track products, transactions, expenses, and overall business performance.

## Goals

The main goal of this project is to create a simple and reliable system that helps sellers:

- Keep track of their products and inventory.
- Register sales and expenses.
- Understand income, costs, and profitability.
- Reduce the friction of manual record keeping.

Project in early development.
